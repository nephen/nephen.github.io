微信公众号/订阅号留言板使用介绍

发表于 2019-01-17 | 更新于: 2019-01-17 | | 本文被围观了 次

前言

由于新申请的微信公众号没有留言功能了，没有留言就无法跟用户进行交互，写出去的文章好像得不到反馈一样，感觉有蛮难受的，所以就自己制作了一个小程序用于留言，来代替原版公众号的留言功能。

另外，留言板是一块自留地，tx也无法抵达，我们是一个社会主义和谐社会，对于不良言论会坚决杜绝，一起维护一个良好的留言环境。

当然你也可以选择买一个可以留言的公众号进行迁移，毕竟有些人注册的比较早是有留言功能的，只是迁移只能由个人->企业，而个人对个人无法进行迁移，所以看起来小程序是暂时唯一的办法了。

本来该小程序考虑完全支持免费入驻，但是由于云端存储有限，没有钱买入大的服务器，所以暂时是前30个可以免费支持接入了。具体入驻方法请在后台回复入驻获取，或者联系wx：DashuTech。

功能特点

参考Ethanm的插件，进行了结合订阅号的定制与完善，目前有如下功能特点。

  • 支持长按删除评论操作，支持配置Admin用户删除其他用户的评论
  • 支持评论点赞功能
  • 支持评论回复功能，即子评论
  • 子评论支持点赞功能
  • 支持emoji表情显示
  • 支持文章阅读量统计功能

使用方法

  1. 在公众号图文编辑末尾插入小程序，点击菜单栏的小程序即可。
  2. 在选择小程序栏输入订阅号留言板进行搜索，搜索到如下图标的小程序，再确认下一步。
  3. 进入填写详细信息栏，小程序路径填写：pages/index/index?scene=你的文章标题&passwd=你的密码，展示方式选择小程序卡片，卡片标题可作为留言引导语比如轻触这里留言~。
  4. 卡片样式处需要上传图片，上传你自己喜欢的就好啦，完成后点击确认，效果如下图。文章发布前建议预览看下效果怎么样。
