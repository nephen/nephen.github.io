公众号怎么开通留言功能——订阅号留言板小程序

发表于 2019-03-03 | 更新于: 2019-04-04 | | 本文被围观了 次

专门为没有留言功能的订阅号设计的留言板

已购置大型服务器，支持大量入驻，入满为止，先到先得！

请在该小程序-“我的”页面点击申请入驻进行入驻，有任何疑问联系wx：DashuTech(添加微信还可以第一时间获取小程序相关通知，拉你进公众号大佬群，可以聊推广经、互粉哦)，欢迎在小程序说明文章中打赏我，同时可以在我爱理生活公众号中回复“留言板”查看留言板使用效果，也欢迎赞赏、看好、转发该文章支持我，优先选择脸熟的小伙伴哦。

留言板介绍与入驻

功能特点

  • 除了作者可以回复留言外，其他人也可以回复(与微信原版不一样的功能)
  • 支持留言板背景图片自定义(与微信原版不一样的功能)
  • 支持精选留言（新功能）
  • 回复留言时被回复者可以接收到服务通知(新功能)
  • 长按留言进行该留言置顶：作者可以置顶任意留言(新功能)
  • 支持作者在消息页面查看所有入驻公众号最新留言(新功能)
  • 支持管理留言板(小程序-“我的”页面点击管理留言板)(新功能)
  • 长按留言可以删除留言：作者可以删除其他用户留言
  • 长按留言进行留言复制(新功能)
  • 支持一个微信号入驻多个公众号，以及多个微信号入驻一个公众号(特殊功能)
  • 支持点赞
  • 支持表情显示
  • 支持文章阅读量统计
  • 支持下拉留言板刷新留言
  • 支持文章作者回复或者留言时昵称为特殊标识的“作者”(新功能)

关于入驻

  • 使用留言板前，请先申请入驻
  • 入驻只是一个说法，入驻成功后，申请入驻的公众号才会拥有新建留言板，查看最新消息，管理留言的权限。
  • 请在该小程序-“我的”页面点击“申请入驻”进行入驻。
  • 提交入驻信息后，请留意“订阅号留言板”的审核结果服务通知，可以关注“我爱理生活”公众号回复“留言板”查看在公众号文章的实际效果，听说赞赏、转发、好看该公众号文章后入驻通过率会更高哦。
  • 入驻成功后，请使用该小程序-“我的”页面-新建留言板来建立你的留言板，并嵌入到公众号文章里面。

设置步骤

(一) 新建留言板

  1. 在小程序-“我的”页面点击新建留言板，填好文章标题、填写背景图片链接（如何获取图片链接参考首页文章说明），以及设置是否精选留言后，点击【一键生成路径】。
  2. 新建留言板成功以后，会跳到新建留言板成功页面，点击【复制路径】。
  3. 新建留言板成功后，点击“查看效果”，可以马上查看留言板效果。还可以将自己的留言板分享给小伙伴们哦。
  4. 如果新建留言板失败，会跳到新建留言板失败页面，失败的原因有两种，一是信息填写不完整，文章标题为必须填写的项；二是还没有进行入驻。

(二) 将留言板嵌入公众号文章

  1. 前往公众号图文编辑插入小程序，点击”小程序”，在“选择小程序”页面中，搜索订阅号留言板，点击“下一步”。

  2. 在“填写详细信息”页面，将新建留言板时复制到的路径（即新建留言板中的步骤二），粘贴至小程序路径处(把默认的路径替换掉)。如果路径没有复制上，也可以在小程序“我的”-管理留言板中重新复制。注意：展示方式如果选择小程序卡片 ，可能由于浏览器原因造成上传卡片失败，可以尝试换个浏览器进行操作。


    说明一下，复制到的路径格式为：pages/board/board?title=你的文章标题&public=你的入驻信息&image=背景图片地址，你的标题建议和要发表的文章标题保持一致，你的入驻信息指的是你入驻时提供的公众号名字，背景图片地址用于在留言列表上方显示背景图，不填则使用默认的背景图片。
  3. 为了防止路径填写错误，强烈建议在发布文章前预览一下，在自己手机上看下文章里面的留言板是否能正常打开，发布以后发现弄错了就没法改了！
  4. 文章发布后，可以在留言板添加阅读原文功能，请移步至-管理留言板-设置原文链接进行（用此功能前请先将公众号关联小程序哦）。
  5. 精选留言管理也在管理留言板的页面里面，开始精选就可以筛选处你认为可以对所有人可见的留言。
