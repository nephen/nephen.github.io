谈谈学习感悟

发表于 2019-03-02 | 更新于: 2019-04-04 | | 本文被围观了 次

最近一直在弄留言板小程序，感觉好长时间没有更新推文了。为了使得用户留言的体验更好，让更多的人拥有留言的功能，小程序也在努力一版一版的完善当中，确实花了不少的时间，一直想更新的推文结果拖到了现在。

这篇来谈谈学习，最近半年混进了几个大佬圈，也学到了不少东西，这里我会把我吸收到的东西二次分享出来，结合我自身的体验谈谈对高效学习的看法。

好的学习态度

高效学习前，要有一个好的学习态度，需要明白一个道理：学习是一件“逆人性”的事，就像锻炼身体一样，需要人持续付出，会让人感到痛苦，并随时想放弃。

健身这个例子非常合适，去年我从4月份开始健身，一直坚持到年底健身房最后关门的一天，健身的一个小时内，着重对需要锻炼的部位加强训练，期间会感觉非常痛苦，很容易就想着放弃，好在健身还是有了点效果，健完身后再加以合理的饮食，去年的增肌目标算是勉强完成了，但是如果每天去健身房打个卡，健身一分钟，拍照半小时，那就结果可想而知了，所以锻炼也是越痛苦，所获得的成就感就越大。

另外一点，不能自律，无法坚持，再好的方法都无法学好，需要明白坚持的重要性。学习本身是一件滞后的事情，不要想着今天学了个东西，期待明天就能给你带来回报，学习需要长期的积累，当你明白复利曲线是什么样的时候，你就知道为什么需要坚持，因为前期的效果很不明显，而当你积累到某一定程度的时候，自然也就到了指数增长开始爆发的时候，我认识的一个大佬，之前一直混技术圈，但是长期以来保持对商业、运营、产品、写作等知识学习的积累，终于在这两年爆发了，今年年初保时捷提回了家。如果不是前期的持续学习积累，公众号红利、知识付费红利开始到来的时候，怎么能抓到这样的好机会呢？

明白主动学习与被动学习

这里说明一下什么是主动学习，什么是被动学习，被动学习包括听讲、阅读、视听、演示，学习内容留存率为5%、10%、20%和30%。而主动学习包括通过讨论、实践、教授给他人，会将原来被动学习的内容留存率从 5% 提升到 50%、75%。

站在程序员的角度来看，你光是看小程序的教程文档来记录学习，跟你利用所学的知识亲手自己搭建一个小程序，所得到的收获明显是不一样的，只看不实践有点像站在岸上学游泳。而如果你能把你所学习的知识和自己的实际体会按照自己的语言分享出去，能把别人说明白的时候，又是另外一个学习的段位了。自己会用和能自己总结归纳出来，是两种不同的境界，总结的过程中，你会有了更多的思考，能够自己去摸索知识间的某些联系脉络，能够做到举一反三，这个时候就开始体会到什么叫主动学习了。

其中，主动学习和系统学习联系比较密切，为了更好的主动学习，让知识有更好的留存率，就要学会去系统的学习。同样站在程序员学习技术的角度看，我们学习一门新技术的时候，就要首先去思考该技术出现的背景、初衷和要达到什么样的目标或是解决什么样的问题，这样我们才能看懂这个技术的设计理念。这个技术的优势和劣势分别是什么？要知道，解决一个问题的时候往往会带来新的问题。技术的使用场景都有哪些？因为离开了场景可能会有很多槽点，没有任何一个技术是普适的。弄清楚技术的组成部分和关键点，毕竟学习技术的核心部分是快速掌握的关键。为了弄清楚以上这些问题点，我们自己就需要花大量的时间主动去思考去探索，最后将所得到的你的理解总结下来，形成自己的知识地图，后续再遇到这些问题或者需要不错说明的地方，再去一一完善这张知识地图，将感性认识提升到理性的认识。知识的起源为哲学，以“作者“心态思考，对知识的来源理解更加深刻。

之所以要学会系统学习，从另外一个方面看待：学习不是努力读更多的书，盲目追求阅读的速度和数量，这会让人产生低层次的勤奋和成长的感觉，这只是在使蛮力。要思辨，要践行，要总结和归纳，否则，你只是在机械地重复某件事，而不会有质的成长的。所以如果让知识的记忆占据了大脑的大部分带宽，可能你思考的时间就少了，当然定期复盘已经学习过的知识是有必要的。

顺便说一下对现在非常盛行的“知识付费”的理解，知识付费确实比自己在网上随便找到的一些公共资源要靠谱，毕竟这里的知识是要收费的，只要有钱赚，那作者的动力就更加足一些，也愿意分享自己的独家秘籍，所以你能学习到的知识肯定更加符合你的需求，这是一个金钱换来了时间的好例子，因为不需要一层一层搭理框架，知识更加系统化了，但是还是需要融入自己的知识体系，千万不要以为知识付费的东西看一遍过一遍你就能真正学到了，只有融入自己学习体系的知识才是真正属于你自己的。知识付费不是学习的终点，而是思考的起点！

总结

学习即分享，正因为自己还有很多需要学习需要弄明白的，学会分享，让知识学的更加深刻。学会分享，才会有更多人愿意与你分享，打破知识的界限，打破知识的诅咒，让认知与知识同步升级。学会分享，形成自己的闭环原则，别人的认可就是你不断前行的正反馈，同时你也会不断扩大自己的影响力，这可能是一笔财富。
