订阅号留言板：公众号留言功能开通新思路！（建议收藏）

发表于 2019-04-04 | 更新于: 2019-04-08 | | 本文被围观了 次

前言

由于新申请的微信公众号没有留言功能了，没有留言就无法跟读者进行互动，写出去的文章好像得不到反馈一样，像一个人的独角戏，感觉有蛮难受的，所以就制作了一个小程序用于留言，来代替原版公众号的留言功能。

当然你也可以选择在a5.net上买一个可以留言的个人公众号进行迁移，毕竟有些人注册的比较早是有留言功能的，只是迁移的话需要你的号是企业号才可以，个人号还是没有希望，所以看起来小程序是暂时唯一的办法了。

本来该小程序考虑完全支持免费入驻，但是无赖于服务器费用太贵，所以暂时是有赞赏、转发、好看等混过脸熟的小伙伴优先入驻了。

需要说明的是，留言板是一块作者的自留地，tx也无法抵达，同时是一个与读者交流的地方，不是一个不良言论的驻留地，所以对于不良言论我们会坚决抵制，一起维护一个良好和谐的留言环境。

功能特点

  • 除了作者可以回复留言外，其他读者可以相互互动(与微信原版不一样的功能)
  • 支持设置菜单栏，收集粉丝意见或者业务需求等(与微信原版不一样的功能)
  • 支持留言板背景图片自定义(与微信原版不一样的功能)
  • 支持精选留言（新功能）
  • 读者留言时运营者能够接收到留言消息提醒（新功能)
  • 回复留言时被回复者可以接收到留言回复通知(新功能)
  • 精选留言时被精选者可以接收到留言入选通知(新功能)
  • 长按留言进行该留言置顶：作者可以置顶任意留言(新功能)
  • 支持作者在消息页面查看所有入驻公众号最新留言(新功能)
  • 支持管理留言板(小程序-『我的』页面点击管理留言板)(新功能)
  • 长按留言可以删除留言：作者可以删除其他用户留言
  • 长按留言进行留言复制(新功能)
  • 支持一个微信号入驻多个公众号，以及多个微信号入驻一个公众号(特殊功能)
  • 支持点赞
  • 支持表情显示
  • 支持文章阅读量统计
  • 支持下拉留言板刷新留言
  • 支持文章作者回复或者留言时昵称为特殊标识的“作者”(新功能)

关于入驻

  • 使用留言板前，请先申请入驻：入驻只是一个说法，入驻成功后，申请入驻的公众号才会拥有新建留言板，查看最新消息，管理留言的权限。
  • 请在该小程序-『我的』页面点击“申请入驻”进行入驻，只有作者才需要入驻，读者不需要入驻即可留言。
  • 提交入驻信息后，请留意“订阅号留言板”的审核结果服务通知。
  • 入驻成功后，请使用该小程序-『我的』页面-新建留言板来建立你的留言板，并嵌入到公众号文章里面。

设置步骤

(一) 新建留言板

  1. 在小程序-『我的』页面点击新建留言板，填好文章标题（背景图片链接先不填，可发布文章后再更换，参考如何更换留言板背景图？），以及设置是否精选留言后(勾选则留言需要经过作者审核后才会对所有人可见)，点击【一键生成路径】。
  2. 新建留言板成功以后，会跳到新建留言板成功页面，点击【复制路径】，通过文件传输助手发给PC端微信。

(二) 将留言板嵌入公众号文章

  1. 前往公众号图文编辑插入小程序，点击”小程序”，在“选择小程序”页面中，搜索订阅号留言板，点击“下一步”。

  2. 在“填写详细信息”页面，将新建留言板时复制到的路径（即新建留言板中的步骤二），粘贴至小程序路径处(把默认的路径替换掉)。如果路径没有复制上，也可以在小程序『我的』-管理留言板中重新复制。
    注意：展示方式如果选择小程序卡片 ，可能由于浏览器原因造成上传卡片失败，可以尝试换个浏览器进行操作。

    说明一下，复制到的路径格式为：pages/board/board?title=你的文章标题&public=你的入驻信息&image=背景图片地址，你的标题建议和要发表的文章标题保持一致，你的入驻信息指的是你入驻时提供的公众号名字，背景图片地址用于在留言列表上方显示背景图，不填则使用默认的背景图片。
    提示：建议在引导语中写明要在留言板里面讨论的议题，这样留言的意愿更强。

  3. 为了防止路径填写错误，强烈建议在发布文章前预览一下，在自己手机上看下文章里面的留言板是否能正常打开，发布以后发现弄错了就没法改了！

  4. 文章发布后，可以在留言板添加阅读原文功能，请移步至-管理留言板-设置原文链接进行（用此功能前请先将公众号关联小程序哦）。
  5. 所谓精选可以简单的理解为筛选，精选留言管理也在管理留言板的页面里面，点击开始精选就可以精选出你认为可以对所有人可见的留言，精选某条留言后，读者会收到留言入选通知，注意：开启精选留言后，最新留言消息需要在该小程序的消息页面查看，直接从推文里面的留言板进去是看不到别人留言的，因为还没精选别人的留言让所有人可见。

如何管理留言板

名称说明：

  1. 从公众号文章里面进入的留言板界面称之为留言板前台，每个读者都可见；
  2. 从小程序消息页面、管理留言板精选页面、以及作者收到的留言回复通知进入的留言板界面称之为留言板后台。

留言板管理需要在手机上进行操作，电脑无法打开小程序。

留言回复

留言板前台或后台界面：点击留言内容即可进行回复，注意不是点击头像或昵称，是留言内容。

留言删除/置顶

留言板前台或后台界面：长按留言内容，会弹出窗口，选择删除或置顶即可，删除后无法恢复，谨慎操作。

精选留言

留言板后台界面：点击精选此留言，在留言板前台即可出现该留言，留言者也会收到留言入选通知。

设置公众号菜单

解决问题一

还在担心链接的素材无法打开留言功能(原版的留言功能也无法打开哦)，在素材里面添加留言板小程序完美解决这个问题！只需在即将发布的文章或素材里面插入留言板即可，具体方法见设置步骤部分。

解决问题二

该菜单可以根据自己的需求设置，比如用于收集粉丝们的意见，或者针对自己的业务征集相关信息等等。

以下介绍了菜单栏和子菜单栏的设置方法，具体如下图片，其中备用网页需要点击下面的从公众号图文消息中选择。

如何更换留言板背景图？

采用公众号素材图片作为背景

  1. 使用你的公众号登录微信公众平台，找到素材管理-图片，点击你想要选择的图片。
  2. 放大图片后，点击查看原图。
  3. 复制浏览器里面的链接，在该小程序-我的页面管理留言板里面进行背景图片设置。

注意事项

看到很多小伙伴还是不太会在公众号里面使用小程序，以为需要关联才能使用，或者以为关联了小程序就能直接插入使用了，其实还是有一些需要注意的地方，想在这里重点提及一下，顺带介绍一下在订阅号文章里面使用订阅号留言板小程序的方法，附实操视频。

  1. 无需关联即可使用小程序。选择小程序时，可以输入小程序的名称即可搜索到，所以只要记得小程序的名称就可以了，如果公众号关联了该小程序的话，那么会在常用里面出现，所以没有关联也是可以使用的。
    毕竟如果一定要关联才能使用小程序，那岂不是很多人都用不了小程序了？因为现在一个小程序只能被500个公众号关联呢！
  2. 需要设置小程序路径。使用任何一个小程序，都需要指定该小程序的路径，就像你想在浏览器打开某个特定的页面，你必须告诉浏览器该页面的网址，所以该小程序的路径就相当于网页的网址。
    拿订阅号留言板小程序来说，默认的小程序路径是pages/home/home，但是你要设定你个人的留言页面，就需要更改该小程序路径，目前是改为pages/board/board后面带参数的格式。
    如果没有更改该路径，那么点开你设置的小程序链接就会跑到该小程序的home页面即首页去，不要问我为什么会这样，就是因为你没有更改路径！

  3. 每篇文章的留言板都需要重新设定小程序路径吗？不一定。如果你每篇文章都可以共用一个留言板，即留言板的标题都是一样的，那就使用之前那个留言板的路径就好了，这样的好处就是，文章刚发布，下面就有留言了，因为留言是接着之前的留言板来的。
